Vocal classification with/without convolution

This was a Practical Work exam during my Master's degree.

We had an audio dataset of 10 classified noises (clock_tick, rooster, crying baby...).
We transform each audio file into a spectrogram image. Then we build two models. The first one is a Multi-Layer Perceptron. The second one is a Convolutional Neural Network.
