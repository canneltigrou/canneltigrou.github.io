Little project using dlib facial detection

Use of the face landmarks from dlib (pretrained model to find 68 landmarks of a face), to build a corresponding smiley.
1. The face is detected (dlib - HOG) with the landmarks,
2. then taking 3 points, the face is rotated to align the eyes and nose : the angle is computed,
3. then the measure between different landmarks are used to build the caracteristics of the smiley,
4. Finally, the smiley face is rotated from the opposite angle.
I plan to use heroku as server to make a website where the user can allow the camera and see the smiley in real time. It is still in construction. Here is the link.
